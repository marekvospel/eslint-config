These are settings for Eslint I use in my projects.

## Installing
Install `eslint` and `eslint-config-vospel`
```shell
npm i -D eslint eslint-config-vospel
```

Add extends to your `.eslintrc`
```js
module.exports = {
  extends: [
    'vospel'
  ]
}
```
